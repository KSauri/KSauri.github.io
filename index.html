Tbili

Inspired by Airbnb, Tbili is a fullstack single page app. It leverages Google's Geocoder and Maps APIs to allow users to create and show new rentals. It heavily incorporates React and Redux, to allow each component access to the state of the app while generating deterministic app renders.

Algorithm Visualization
FlightMapper

FlightMapper is a visualization of the A* path-finding algorithm. Using map data on the 250 largest airports in the US, it demonstrates exhaustive search as well as the efficiency of A*. It uses plain JavaScript and cascading callback functions to illustrate paths checked by the algorithm.
